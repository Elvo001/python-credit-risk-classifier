# Python Credit Risk Classifier

A credit risk classification pipeline built in Python, demonstrating probability-of-default modeling, threshold-based decisioning, and evaluation using industry-relevant metrics.

---

## 🚀 Overview

This project simulates a real-world credit risk modeling workflow:

- Input: structured applicant-style features  
- Output: probability of default (PD)  
- Decision: threshold-based approve / reject classification  

---

## 🧠 Key Concepts

- Logistic regression baseline (governance-friendly)
- Probability estimation (PD modeling)
- Threshold-based decisioning
- Evaluation using:
  - ROC-AUC
  - Gini coefficient
  - KS statistic
- Confusion matrix & classification report
